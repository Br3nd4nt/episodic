# Installation

## Requirements

- Python 3.7+
- Internet connection

## Install as CLI Tool

```bash
git clone https://github.com/your-username/episodic.git
cd episodic
pip install -e .
```

## Test Installation

```bash
episodic -h
```

## Development Mode

```bash
python -m venv venv
source venv/bin/activate
```

## First Run

```bash
episodic -p /path/to/your/episodes -s "Your Show Name" -n 1 -v
```

## Uninstall

```bash
pip uninstall episodic
```

That's it!
